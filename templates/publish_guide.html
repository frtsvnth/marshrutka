YouTube — настройка публикации

1. Что понадобится

  • Аккаунт Google (YouTube-канал)
  • Проект в Google Cloud Console с включённым YouTube Data API v3
  • OAuth 2.0 Client ID (тип «Веб-приложение»)

2. Настройка Google Cloud Console

  1. Откройте Google Cloud Console
  2. Создайте проект → APIs & Services → Library → включите YouTube Data API v3
  3. Перейдите в OAuth consent screen, выберите External, заполните обязательные поля, добавьте свой email в Test users
  4. Перейдите в Credentials → Create Credentials → OAuth client ID
  5. Тип: Web application
  6. В Authorized redirect URIs добавьте: http://localhost:9090/publish/youtube/callback
  7. Сохраните → скачайте JSON

3. Создание профиля в Marshrutka

  1. Откройте /publish/profiles/new
  2. Выберите платформу YouTube
  3. Название: YouTube — Зад Пегаса (или другое)
  4. В поле Credentials JSON вставьте: {"client_id": "xxx.apps.googleusercontent.com", "client_secret": "GOCSPX-..."}
  5. Сохраните профиль

4. OAuth-авторизация

  1. Откройте страницу редактирования профиля
  2. Нажмите 🔑 Авторизовать YouTube
  3. Подтвердите доступ в Google
  4. Если у аккаунта один канал — профиль готов. Если несколько — выберите нужный канал
  5. Статус сменится на ✓ готов к публикации

5. Привязка к проекту

На странице редактирования проекта выберите профиль в разделе Publish profiles.

6. Публикация

  1. Запустите pipeline
  2. На странице запуска (после успеха) нажмите 📤 Создать запрос
  3. Нажмите ▶ Выполнить
  4. Видео загрузится в YouTube

7. Несколько каналов на одном Google-аккаунте

Marshrutka использует prompt=consent, поэтому каждый OAuth-заход даёт отдельный refresh_token. Вы можете авторизовать 3 разных профиля для 3 разных каналов на одной учётной записи:

  1. Создайте профиль «YouTube — Зад Пегаса» → авторизуйте → выберите канал 1
  2. Создайте профиль «YouTube — Ежу Понятно» → авторизуйте → выберите канал 2
  3. Создайте профиль «YouTube — Сапоги-сандалики» → авторизуйте → выберите канал 3

8. Приватность по умолчанию

В настройках профиля можно выбрать видимость: public, unlisted или private.
