YouTube Shorts — настройка публикации

1. Что понадобится

  • Аккаунт Google (YouTube-канал)
  • Проект в Google Cloud Console
  • Включённый YouTube Data API v3
  • OAuth 2.0 Client ID (Desktop или Web)

2. Какие доступы нужны

  • ✓ готово YouTube Data API — загрузка видео
  • скоро YouTube Analytics API — сбор статистики
  • вручную OAuth consent screen — настройка в Google Cloud

3. Поля профиля

  • Название профиля — для удобства (например «YouTube — основной»)
  • Credentials JSON — содержимое файла client_secret.json из Google Cloud
  • Канал — название и ID (заполняются автоматически после привязки)
  • Шаблоны — заголовок, описание, теги для Shorts
  • Видимость — public / unlisted / private

4. Как проверить готовность

После заполнения профиля на странице профилей появится статус ✓ готов к использованию. Если credentials не заполнены или некорректны — будет требует настройки.

5. Привязка к проекту

На странице редактирования проекта выберите разрешённые профили публикации из списка. Укажите профиль по умолчанию.

6. Запуск публикации

После успешного завершения pipeline на странице запуска нажмите «Опубликовать». Выберите профиль, при необходимости отредактируйте заголовок/описание и подтвердите.

7. Что поддерживается

  • ✓ готово Интерфейс профилей и настройки
  • ✓ готово Привязка профилей к проектам
  • скоро Автоматическая OAuth-авторизация
  • скоро Реальная загрузка видео через API
  • вручную Настройка OAuth consent screen в Google Cloud

Instagram Reels — настройка публикации

1. Что понадобится

  • Аккаунт Instagram (профессиональный или creator)
  • Связанная Facebook-страница
  • Приложение в Meta for Developers
  • Instagram Graph API доступ

2. Какие доступы нужны

  • скоро Instagram Graph API — публикация Reels
  • вручную Настройка приложения в Meta for Developers
  • вручную Получение долгоживущего access_token

3. Поля профиля

  • Название профиля — например «Instagram — Reels»
  • Credentials JSON — access_token и app_id
  • Аккаунт — Instagram business account ID
  • Шаблоны — подпись, хештеги для Reels

4. Как проверить готовность

После заполнения базовых полей статус сменится на «готов». Полноценная публикация через API появится позже.

5. Привязка к проекту

На странице редактирования проекта добавьте Instagram-профиль в список разрешённых.

6. Запуск публикации

В настоящее время публикация в Instagram — в режиме «скоро». Интерфейс готов, API-адаптер будет добавлен в одном из следующих обновлений.

7. Что поддерживается

  • ✓ готово Интерфейс профилей и настройки
  • ✓ готово UI для создания профиля Instagram
  • скоро API-адаптер для публикации Reels
  • вручную Настройка Meta-приложения и токенов

VK Видео — настройка публикации

1. Что понадобится

  • Аккаунт ВКонтакте
  • Standalone-приложение VK (созданное в dev.vk.com)
  • Access token с правами video
  • ID сообщества или пользователя для публикации

2. Какие доступы нужны

  • скоро VK API video.save — загрузка клипов
  • вручную Создание приложения на dev.vk.com
  • вручную Получение access_token (implicit flow)

3. Поля профиля

  • Название профиля — например «VK — Клипы»
  • Credentials JSON — access_token и app_id
  • ID сообщества — отрицательное число для группы
  • Шаблоны — название и описание для клипов

4. Как проверить готовность

Профиль считается готовым, если указан access_token. Полноценная загрузка видео через VK API будет добавлена позже.

5. Привязка к проекту

Добавьте VK-профиль в проект на странице редактирования.

6. Запуск публикации

UI готов. API-адаптер для VK Video будет реализован в ближайших обновлениях.

7. Что поддерживается

  • ✓ готово Интерфейс профилей VK
  • ✓ готово UI для настройки
  • скоро API video.save + video.get
  • вручную Создание приложения и токен
