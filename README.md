# marshrutka

**Панковый раннер контент-маршрутов.**  
Минималистичный project runner для контентных пайплайнов: проекты, ordered jobs, runs, cron и публикация из одного окна + AI-агент Маршал в правой панели.

Marshrutka — это лёгкий раннер для контентных проектов. Он хранит проекты, их входные формы, ordered job-цепочки, статусы run'ов и расписания, а потом довозит результат до YouTube через отдельный publish-job. Встроенный AI-агент помогает писать тексты, анализировать проекты, управлять расписаниями, искать информацию и редактировать файлы.

## Быстрый старт

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

cp .env.example .env  # настроить API ключи
python main.py
# → http://localhost:9090
```

### Переменные окружения (.env)

```bash
ROUTERAI_BASE_URL=https://routerai.ru/api/v1
ROUTERAI_API_KEY=sk-your-key
ROUTERAI_MODEL=deepseek/deepseek-v4-flash
SERPAPI_API_KEY=your_serpapi_key     # опционально
STT_HTTP_URL=http://.../transcribe   # опционально
```

## API

### Marshrutka core

| Метод | Путь | Описание |
|---|---|---|
| `GET` | `/api/projects` | Список проектов |
| `GET` | `/api/projects/{id}` | Детали проекта |
| `POST` | `/api/projects` | Создать проект |
| `POST` | `/api/runs` | Создать запуск |
| `GET` | `/api/runs` | Список запусков |
| `GET` | `/api/runs/{id}` | Детали запуска |
| `GET` | `/api/runs/{id}/logs` | Логи запуска |
| `GET` | `/api/schedules` | Список расписаний |
| `POST` | `/api/schedules` | Создать расписание |
| `DELETE` | `/api/schedules/{id}` | Удалить расписание |
| `GET` | `/api/health` | Проверка здоровья |

### AI-агент Маршал

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/agent/chat` | Отправить сообщение (SSE stream) |
| `GET` | `/api/agent/context` | Контекст системы + сессии |
| `POST` | `/api/agent/memory` | Обновить профиль |
| `POST` | `/api/agent/memory-tool` | Инструменты памяти |
| `POST` | `/api/agent/state/save` | Сохранить состояние UI |
| `GET` | `/api/agent/state/load` | Загрузить состояние UI |
| `POST` | `/api/agent/transcribe` | Распознавание речи |
| `DELETE` | `/api/agent/history` | Очистить историю |

Локальный AI-агент с 24 инструментами: тексты, проекты, расписания, веб-поиск, файлы, память, удаление запусков. Подробнее: `AI_DOCS.md`.

## UI

- **`/`** — список проектов
- **`/projects/new`** — создание нового проекта
- **`/projects/{id}`** — форма запуска, расписания, задачи, последние запуски
- **`/runs/{id}`** — детали запуска с логами и артефактами
- **`/schedules`** — все cron-расписания
- **Правая панель** — AI-агент Маршал на всех страницах

## Дизайн и бренд

Marshrutka — это не enterprise dashboard. Это лёгкий, дружелюбный инструмент с характером:
- **Светлый** и воздушный интерфейс
- **Индиговый** левый сайдбар
- **Тёмная правая панель** с AI-агентом Маршал
- Мягкие скругления и аккуратные тени
- Русский язык интерфейса
- Agent-ready: 24 OpenAI function-calling инструмента
- Диалог с агентом переживает обновление страницы (localStorage)

## Принципы

- **MVP-first**: минимальный scope, итеративные улучшения
- **Agent-ready**: стабильные routes, понятные статусы, простые формы
- **Нет enterprise-монстра**: без DAG-движка, event bus, multi-tenancy
- **Внешние пайплайны**: оркестратор вызывает существующие конвейеры по API
- **Лёгкое хранилище**: JSON-файлы — легко инспектировать, бекапить и автоматизировать
